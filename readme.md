sis_asfenglish
=======================
Baseadas em vídeo aulas da Universidade XTI

Este simples sistema conta com a leitura e escrita de arquivos de texto
Mas também está integrado ao banco de dados MySQL.

Seu objetivo principal é ajudar na realização de traduções de texto da língua inglesa de forma pedagógica.

---------------------------------------------------------------------------------

Video-based classes at the University XTI

This simple system has the reading and writing text files
But it is also integrated into the MySQL database.

Its main objective is to help in making English language text translations of educational way.
